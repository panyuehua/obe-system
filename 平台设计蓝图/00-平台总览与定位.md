# OBE教管一体化平台 — 产品设计蓝图

> 版本：v1.0 | 日期：2026-04-02 | 状态：设计阶段

---

## 一、平台定位

本平台**不是**业务执行系统，而是一个**数据汇聚与智能分析中枢**，扮演"粘合剂"角色：

```
┌─────────────────────────────────────────────────────────────┐
│                    教务系统（Academic Affairs）               │
│     专业培养方案 / 学生信息 / 课程开设 / 成绩数据              │
└──────────────────────────┬──────────────────────────────────┘
                           │  结构化数据同步
                           ▼
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│          OBE 教管一体化平台（本平台）                         │
│                                                             │
│   ┌──────────────┐  ┌──────────────┐  ┌─────────────────┐  │
│   │ OBE体系构建  │  │ 矩阵计算引擎 │  │  AI智能分析层   │  │
│   └──────────────┘  └──────────────┘  └─────────────────┘  │
│                                                             │
└──────────────────────────┬──────────────────────────────────┘
          ▲                │                    ▲
          │ 课程执行数据    │ 反馈建议            │ 行业数据
          │                ▼                    │
┌─────────┴──────────┐              ┌──────────┴──────────────┐
│  教学平台（LMS）    │              │  外部行业数据源           │
│  课程目标/任务/     │              │  招聘平台 / 政策文件 /   │
│  学习记录/成绩      │              │  行业标准 / 职业认证      │
└────────────────────┘              └─────────────────────────┘
```

### 核心价值主张


| 维度       | 说明                     |
| -------- | ---------------------- |
| **连接价值** | 打通教务系统与教学平台的数据孤岛       |
| **计算价值** | 自动化完成OBE四层达成度矩阵计算      |
| **诊断价值** | 自动识别培养方案中的逻辑缺陷与薄弱环节    |
| **决策价值** | 基于行业数据的AI辅助专业建设决策      |
| **认证价值** | 支持工程教育认证、职业教育评估等多类认证场景 |

## 二、平台用户角色与核心诉求

```
角色层级
├── 学校层
│   ├── 教务管理员      → 系统配置、数据同步、全校概览
│   └── 教学质量督导    → 专业达成度监控、质量预警、评估报告
│
├── 学院/专业层
│   ├── 学院管理员      → 院级数据管理、专业横向对比
│   ├── 专业负责人      → 培养方案设计、矩阵配置、持续改进
│   └── 课程负责人      → 课程目标设计、支撑关系配置
│
├── 教学执行层
│   └── 任课教师        → 成绩录入、资源管理、教学改进
│
└── 学习主体层
    └── 学生            → 个人能力画像、学业预警、达成进度
```

## 三、平台整体功能模块概览

```
OBE教管一体化平台
│
├── M1 数据接入中枢          ← 平台基础设施层
│   ├── 教务系统集成
│   ├── 教学平台集成
│   └── 外部数据接入
│
├── M2 OBE体系配置           ← 核心业务层（设计态）
│   ├── 学校定位与核心能力
│   ├── 培养目标管理
│   ├── 毕业要求管理
│   └── 课程体系管理
│
├── M3 课程OBE设计           ← 核心业务层（课程态）
│   ├── 课程目标（ILO）配置
│   ├── 支撑关系配置
│   └── 考核方式设计
│
├── M4 矩阵计算引擎          ← 核心计算层
│   ├── 四层达成度计算
│   ├── 逻辑冲突诊断
│   └── 持续改进追踪
│
├── M5 可视化分析看板        ← 数据呈现层
│   ├── 专业图谱五维视图
│   ├── 达成度多维分析
│   └── 角色化仪表盘
│
└── M6 AI智能建议层          ← 智能增强层
    ├── 行业匹配度分析
    ├── 培养方案优化建议
    └── 政策导向解读
```
