## 三、四层达成度计算逻辑

```
【设计方向：自上而下】          【计算方向：自下而上】

培养目标(T)                    培养目标达成度
   │  支撑强度(s)                    ▲
   ▼                                │ 加权汇总
一级毕业要求(R1)               一级毕业要求达成度
   │  权重(w1)                       ▲
   ▼                                │ 加权汇总
二级指标点(R2)                 二级指标点达成度
   │  支撑权重(w2)                    ▲
   ▼                                │ 加权汇总
课程(C) ─── 开设 ──→ 教学班      课程目标达成度
                         │              ▲
                         │ 对应         │ 加权汇总
                         ▼
                    课程目标/ILO    ILO达成度
                         │              ▲
                         │ 映射         │ 按权重计算
                         ▼
                      考核任务      学生任务成绩
                    (作业/测验/          │
                     期末/实验)          │ 来自LMS/教务
                                        │
                               ─────────┘
                               原始成绩数据


计算公式链：
─────────────────────────────────────────────────────
ILO_i达成度     = Σ(任务成绩_j × 题目权重_j) / 满分
                  (j为该ILO对应的所有考核题目)

课程目标_k达成度 = Σ(ILO_i达成度 × ILO权重_i) / Σ(ILO权重)
                  (i为支撑该课程目标的所有ILO)

R2_m达成度      = Σ(课程目标_k达成度 × 支撑权重_k) / Σ(支撑权重)
                  (k为支撑该指标点的所有课程目标)

R1_n达成度      = Σ(R2_m达成度 × R2权重_m) / Σ(R2权重)
                  (m为该一级要求下的所有二级指标点)

T_p达成度       = Σ(R1_n达成度 × 支撑强度_n) / Σ(支撑强度)
                  (n为支撑该培养目标的所有一级要求)
─────────────────────────────────────────────────────
```

## 五、持续改进（CQI）业务流程

```
学期结束
   │
   ▼
自动触发达成度计算
   │
   ├── 达标（≥ 阈值）──→ 归档本期数据 ──→ 与上期对比 ──→ 结案
   │
   └── 未达标（< 阈值）
           │
           ▼
       诊断报告生成
       ├── 哪些指标点未达标？
       ├── 原因初步分析（支撑不足/权重偏差/教学问题）
       └── AI改进建议参考
           │
           ▼
       课程负责人填写改进计划
       ├── 改进内容描述
       ├── 预计改进完成期限
       └── 改进措施分类（教学内容/考核设计/教学方法）
           │
           ▼
       专业负责人审核改进计划
           │
           ▼
       下学期执行改进措施
           │
           ▼
       下期达成度计算（验证改进效果）
           │
           ▼
       改进效果评估（改进前后对比）
           │
           └── 写入年度专业持续改进报告
```
