# OBE教管一体化平台 — 功能架构图

---

## 完整功能树

```
OBE教管一体化平台
│
├─────────────────────────────────────────────────────
│  M1  数据接入中枢
├─────────────────────────────────────────────────────
│   ├── 1.1 教务系统集成
│   │   ├── 学校/学院/专业基础数据同步
│   │   ├── 学生信息同步（年级、班级、学号）
│   │   ├── 课程开设数据同步（教学班、任课教师）
│   │   ├── 培养方案数据同步（课程计划、学分）
│   │   └── 成绩数据双向同步（拉取+回写）
│   │
│   ├── 1.2 教学平台（LMS）集成
│   │   ├── 课程任务数据采集
│   │   ├── 学生学习行为数据采集
│   │   ├── 作业/测验/考试成绩采集
│   │   └── 资源使用情况数据采集
│   │
│   ├── 1.3 外部行业数据接入
│   │   ├── 招聘平台数据接口（智联/猎聘等API）
│   │   │   ├── 岗位JD结构化抓取
│   │   │   ├── 技能需求词频分析
│   │   │   └── 薪资级别与岗位热度
│   │   ├── 政策文件智能抓取
│   │   │   ├── 定向政府官网巡检
│   │   │   ├── 政策关键词提取
│   │   │   └── 人才需求研判报告生成
│   │   └── 行业标准与认证体系数据
│   │
│   └── 1.4 数据质量管理
│       ├── 数据同步状态监控
│       ├── 异常数据告警
│       ├── 数据清洗与去重规则配置
│       └── 同步日志审计
│
├─────────────────────────────────────────────────────
│  M2  OBE体系配置（专业培养方案层）
├─────────────────────────────────────────────────────
│   ├── 2.1 学校通用能力框架
│   │   ├── 学校办学定位维护
│   │   ├── 学校级核心能力集定义
│   │   └── 能力框架版本管理
│   │
│   ├── 2.2 专业培养目标管理
│   │   ├── 培养目标条目维护（支持多届版本）
│   │   ├── 培养目标子目标拆解
│   │   ├── 目标-社会需求关联标注
│   │   └── 培养方案版本对比
│   │
│   ├── 2.3 毕业要求管理
│   │   ├── 一级毕业要求定义
│   │   ├── 二级毕业要求（指标点）定义
│   │   ├── 指标点权重配置
│   │   ├── 毕业要求与核心能力映射
│   │   └── 毕业要求标准阈值设置
│   │
│   ├── 2.4 课程体系管理
│   │   ├── 课程基本信息维护
│   │   │   ├── 课程编号/名称/学分/学时
│   │   │   ├── 课程性质（必修/选修/实践）
│   │   │   └── 先修课程关联
│   │   ├── 课程归组管理（认证课组）
│   │   ├── 核心课程标记
│   │   └── 课程开设学期规划
│   │
│   └── 2.5 支撑关系配置
│       ├── 培养目标 → 毕业要求 关联强度配置（强/中/弱）
│       ├── 毕业要求指标点 → 课程 支撑权重配置
│       └── 支撑关系合理性检查（权重求和校验）
│
├─────────────────────────────────────────────────────
│  M3  课程OBE设计（课程教学层）
├─────────────────────────────────────────────────────
│   ├── 3.1 课程目标（ILO）管理
│   │   ├── 预期学习结果（ILO）定义
│   │   ├── ILO与毕业要求指标点关联配置
│   │   ├── ILO权重分配
│   │   └── 知识点挂载（ILO → 知识点树）
│   │
│   ├── 3.2 教学设计配置
│   │   ├── 教学内容安排（周次/学时/知识点）
│   │   ├── 教学方法策略标记
│   │   ├── 教学资源挂载
│   │   └── 数字资产管理（视频/案例/习题/实验）
│   │
│   ├── 3.3 考核方案设计
│   │   ├── 考核方式定义（作业/测验/期末/实验）
│   │   ├── 各考核项权重配置
│   │   ├── 考核项 → ILO 映射关系配置
│   │   └── 达成评价标准设置（合格线）
│   │
│   └── 3.4 教学大纲生成
│       ├── 标准化大纲模板配置
│       ├── 自动生成课程教学大纲
│       └── 大纲版本管理与导出
│
├─────────────────────────────────────────────────────
│  M4  矩阵计算引擎
├─────────────────────────────────────────────────────
│   ├── 4.1 四层达成度自动计算
│   │   ├── 任务/考核 → ILO达成度计算
│   │   ├── ILO → 课程目标达成度计算
│   │   ├── 课程目标 → 毕业要求达成度计算
│   │   └── 毕业要求 → 培养目标达成度计算
│   │
│   ├── 4.2 矩阵可视化
│   │   ├── 培养目标-毕业要求关联矩阵
│   │   ├── 毕业要求-支撑课程矩阵
│   │   ├── 课程-课程目标达成矩阵
│   │   └── 课程目标-考核任务矩阵
│   │
│   ├── 4.3 逻辑冲突自动诊断
│   │   ├── 支撑孤岛检测（无课程支撑的指标点）
│   │   ├── 权重失衡预警（支撑强度不足阈值）
│   │   ├── 知识点冗余检测（跨课程重叠率）
│   │   ├── 真空指标诊断（有目标无知识点支撑）
│   │   └── 达成度未达标预警（低于合格线）
│   │
│   └── 4.4 持续改进追踪
│       ├── 课程改进意见记录（教师/督导/学生反馈）
│       ├── 改进措施执行状态追踪
│       ├── 改进效果跨期对比
│       └── 专业年度持续改进报告生成
│
├─────────────────────────────────────────────────────
│  M5  可视化分析看板
├─────────────────────────────────────────────────────
│   ├── 5.1 专业图谱五维视图
│   │   ├── 专业-课程-能力-知识-资源 五级联动图谱
│   │   ├── 图谱健康度指标（逻辑完备性/资源饱和度/任务闭环率）
│   │   ├── 节点下钻交互（点击展开详情）
│   │   └── 图谱对比（不同年度/不同专业）
│   │
│   ├── 5.2 达成度多维分析看板
│   │   ├── 专业整体达成度雷达图
│   │   ├── 毕业要求分项达成度趋势（多届对比）
│   │   ├── 课程达成度排行与分布
│   │   └── 学生个体达成度分布分析
│   │
│   ├── 5.3 教师教学画像
│   │   ├── 承担课程健康度全景视图
│   │   ├── 教学风格偏好分析（资源型/互动型/技术型）
│   │   ├── 近三学期成长趋势曲线
│   │   └── 资源影响力（跨专业被引用情况）
│   │
│   ├── 5.4 学生学习画像
│   │   ├── 个人能力达成进度（闯关式可视化）
│   │   ├── 学业预警与风险提示
│   │   ├── 学习投入与能力提升关联分析
│   │   └── 毕业能力达成轨迹时间轴
│   │
│   └── 5.5 问卷评价管理
│       ├── 问卷模板库（学生评教/教师自评/社会评价）
│       ├── 问卷发布与定向分发
│       ├── 回收率监控与催收
│       └── 满意度分析报告自动生成
│
├─────────────────────────────────────────────────────
│  M6  AI智能建议层
├─────────────────────────────────────────────────────
│   ├── 6.1 行业匹配度分析
│   │   ├── 校内能力指标 vs 行业岗位需求对比
│   │   ├── 溢出项识别（校内有、行业无需求）
│   │   ├── 缺失项识别（行业热需、校内未覆盖）
│   │   └── 匹配度趋势变化（季度/年度对比）
│   │
│   ├── 6.2 培养方案优化建议
│   │   ├── 基于行业数据的课程增减建议
│   │   ├── 毕业要求指标点修订建议
│   │   ├── 知识点更新优先级排序
│   │   └── 建议可接受/驳回/暂存操作（人工审核）
│   │
│   ├── 6.3 政策导向解读
│   │   ├── 行业政策自动抓取与摘要
│   │   ├── 政策与专业关联度评分
│   │   └── 《人才需求政策研判单》自动生成
│   │
│   └── 6.4 认证辅助
│       ├── 认证指标自查清单自动生成
│       ├── 自评报告关键数据自动填充
│       └── 认证问题点预判与整改提示
│
└─────────────────────────────────────────────────────
   M7  系统管理
└─────────────────────────────────────────────────────
    ├── 7.1 用户与权限管理
    │   ├── 用户账号管理（与教务系统SSO集成）
    │   ├── 角色权限矩阵配置
    │   └── 数据权限控制（学院/专业/个人级别）
    │
    ├── 7.2 基础配置
    │   ├── 学年学期管理
    │   ├── 数据字典维护
    │   ├── 达成度计算方法配置（均值法/加权法等）
    │   └── 通知消息模板管理
    │
    └── 7.3 系统运维
        ├── 数据同步任务调度
        ├── 系统日志与审计
        └── 备份与数据导出
```
